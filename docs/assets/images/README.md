# 画像置き場（スクショ用）

このフォルダは、教材内で使う **スクリーンショット** を置く場所です。

## GitHub ヘッダー解説で使う画像

- `github-global-header.png`: GitHub 全体の上部ヘッダー（github.com の上部）
- `github-repo-tabs.png`: リポジトリのタブ（Code / Issues / Pull requests などが並ぶ部分）
- `github-global-header-logged-in.png`: GitHub 全体の上部ヘッダー（ログイン後。PR/Issues/Notifications/プロフィール等が見える状態）

このリポジトリには **例として（ログイン前の）スクショ**を同梱しています。
自分の画面に合わせたい場合は、この2ファイルを自分のスクショで置き換えてください。

画像を置いたら `docs/git-github/github-header.md` の番号（ピン）の位置を調整してください。
